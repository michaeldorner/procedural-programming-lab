## Running Tests

Run tests for a single exercise:

```bash
cd exercise_01
python -m unittest discover tests -v
```

Run all exercises:

```bash
python run_tests.py
```

## Notes

- Follow PEP8 styleguidelines.
- Do not print the result inside the function.
- Do not change the function signatures.
